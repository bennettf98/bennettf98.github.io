# Disability Web Accessibility

## About this Project

This site was created for an academic group project, written with HTML and Markdown using a Hugo theme for styling. Website contains several text entries providing information on the importance of web disability access and various examples of beneficial tools to improve accessibility.

## How it Works

Users can navigate throughout the site to view text entries on why web accessbility is important and why certain examples of accessbility tools are important and useful themselves. Users can also navigate to text entries showcasing six different text entries containing examples of different web accessiblity tools, such as physical ones such as keyboard navigation tools, or digital ones such as screen readers, alternative text, and resizable text.

## Built With

* Markdown
* HTML
* Hugo

## How to Run

This project simply exists as a website accessible to anyone on the internet, no downloads or outside programs necessary.
